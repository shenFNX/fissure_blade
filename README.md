## 📖 模组介绍 (Introduction)

**裂缝之刃** 添加了一把极其强大的武器，它不仅拥有毁灭性的伤害，还拥有撕裂现实、复制实体的诡异能力。

当这把剑斩杀生物时，现实的裂缝会将目标“一分为二”。被击杀的生物不会彻底死亡，而是会**分裂成两个生命值减半的复制体**。

### ✨ 主要特性 (Features)

*   **💥 毁灭打击**: 
    *   对非玩家生物造成 **10000点** 基础伤害。
    *   面板攻击力: **233** (一击必杀绝大多数生物)。
    *   攻速较慢 (类似于斧头)。

*   **🦠 裂变诅咒 (The Fissure)**:
    *   当你使用裂缝之刃击杀生物时，该生物会**分裂**成两个。
    *   **分裂规则**:
        *   分裂出的新个体拥有原版 **50% 的最大生命值**。
        *   分裂体依然可以被再次分裂，直到生命值过低（低于 2 点）无法维持形态，此时它们会化为虚空烟雾彻底消散。
    *   **Boss 限制**: 凋灵、末影龙和监守者等 Boss 生物**只能分裂一次**，无法无限增殖。
    *   **玩家豁免**: 该效果对玩家无效。
